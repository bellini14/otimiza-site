# Home Hero Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Recriar a hero da home com uma nova identidade brutalista minimalista, CTA central e fundo reativo ao mouse.

**Architecture:** A implementação fica concentrada em `src/pages/Home.jsx` para a estrutura e comportamento de ponteiro, com apoio de `src/index.css` para tokens visuais, camadas e animações. Um teste de interface em `src/pages/Home.test.jsx` cobre o conteúdo principal e a atualização das custom properties do palco interativo.

**Tech Stack:** React 19, React Router, Tailwind CSS, Vitest, Testing Library

---

### Task 1: Cobrir a nova hero com teste

**Files:**
- Create: `src/pages/Home.test.jsx`
- Modify: `src/pages/Home.jsx`

- [ ] **Step 1: Write the failing test**

```jsx
it('renders the brutalist hero with centered CTA and pointer-reactive stage', () => {
  render(
    <MemoryRouter>
      <Home />
    </MemoryRouter>,
  )

  expect(screen.getByRole('heading', { name: 'Decidir melhor agora, porque não?' })).toBeInTheDocument()
  expect(screen.getByRole('link', { name: 'Agendar diagnostico' })).toBeInTheDocument()
  expect(screen.getByRole('link', { name: 'Conhecer a consultoria' })).toBeInTheDocument()

  const stage = screen.getByTestId('hero-stage')
  fireEvent.mouseMove(stage, { clientX: 420, clientY: 180 })

  expect(stage.style.getPropertyValue('--pointer-x')).toBe('420px')
  expect(stage.style.getPropertyValue('--pointer-y')).toBe('180px')
})
```

- [ ] **Step 2: Run test to verify it fails**

Run: `npm test -- src/pages/Home.test.jsx`
Expected: FAIL because the old hero copy and `hero-stage` do not exist.

- [ ] **Step 3: Write minimal implementation**

Implement the new hero structure in `src/pages/Home.jsx`, add pointer state and expose the interactive stage with `data-testid="hero-stage"`.

- [ ] **Step 4: Run test to verify it passes**

Run: `npm test -- src/pages/Home.test.jsx`
Expected: PASS

- [ ] **Step 5: Commit**

```bash
git add src/pages/Home.jsx src/pages/Home.test.jsx
git commit -m "feat: redesign homepage hero"
```
