# Inspire Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the old `Insights e Blog` experience with a new `/inspire` editorial area that has its own shell, routing, and Medium-like landing page layout.

**Architecture:** Add a dedicated editorial layout for `/inspire` routes, move the existing blog feed into a new `Inspire` page built from custom editorial components, and retarget post detail pages to the same shell. Keep the existing post data source and Sanity integration, but remap links and presentation to the new route family.

**Tech Stack:** React, React Router, Vitest, Testing Library, Tailwind, custom CSS
